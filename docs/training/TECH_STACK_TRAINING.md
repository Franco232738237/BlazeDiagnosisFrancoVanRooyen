# Blaze Diagnostics Tech Stack Training

## Current Repository Stack

The current repository contains:

- TypeScript
- Next.js
- React
- Tailwind CSS setup added for student training
- Node.js backend structure
- PostgreSQL database design
- Prisma ORM in the existing backend
- GitHub for version control

## Important ORM Note

Earlier planning referenced Drizzle ORM. The uploaded repository currently uses Prisma. Students should learn the actual repository structure first. Mentors should confirm whether database work will continue in Prisma or whether a Drizzle migration will be planned separately.

See `docs/technical/ORM_DECISION_NOTE.md`.

## What Software Students Should Learn First

### TypeScript

Focus on:

- types
- interfaces
- function parameters
- return types
- optional fields
- arrays and objects

### React

Focus on:

- components
- props
- state
- hooks
- event handling
- rendering lists
- conditional rendering

### Next.js

Focus on:

- app router basics
- pages and layouts
- client components vs server components
- project folder structure
- environment variables

### Tailwind CSS

Focus on:

- spacing classes
- layout classes
- typography classes
- borders and rounded corners
- responsive classes
- building simple cards, forms, and dashboards

### Backend

Focus on:

- controllers
- services
- repositories
- validators
- DTOs
- middleware
- API route contracts

### Database

Focus on:

- PostgreSQL tables
- primary keys
- foreign keys
- relationships
- migrations
- ORM schema files
